# Webカメラ物体検出デモ（COCO-SSD）

ブラウザ上で動作するシンプルな物体検出デモです。カメラ映像に対してCOCO-SSDで検出した枠とラベルを描画します。

## スクリーンショット

![デモ画面](スクリーンショット%202026-02-10%20150102.png)

## 使い方

1. `index.html` をHTTPSで配信できる場所（例: GitHub Pages）に置きます。
2. ブラウザで開き、「カメラ開始」を押して権限を許可します。

## 仕様

- モデル: COCO-SSD (TensorFlow.js)
- 対応: デスクトップChrome/Edge推奨（iPhone/Safariは性能差が出る場合あり）
